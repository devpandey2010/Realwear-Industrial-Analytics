**Business Objectives**

1. Which plant has the highest operational downtime and command failure rate?
Compare all 5 plants to identify which location needs the most urgent intervention.

2. Does ambient noise level significantly affect voice command accuracy and failure rate?
Determine if high-noise environments (above 85 dB) cause more command failures — and by how much.

3. Which communication platform performs better — MS Teams or Webex?
Compare session completion rate, resolution time, command success rate, and productivity score across both platforms.

4. How does shift timing (Morning / Afternoon / Night) affect worker productivity and issue resolution?
Identify if night shift workers perform worse and whether shift scheduling needs to change.

5. Which departments and worker roles have the highest unresolved issue rate?
Pinpoint departments where issues are frequently dropped or unresolved to target training or resource allocation.

6. How is device health degrading over time, and which devices are at highest risk of failure?
Track battery drain, health scores, and usage hours across all 20 RealWear devices to flag devices needing maintenance.

7. Does using an external microphone reduce command failures in high-noise environments?
Statistically test whether external mic usage improves command success rate — and recommend a policy.

8. Can we predict the number of command failures for a given session before it happens?
Build a regression model using noise level, platform, connection type, and shift to predict command failures in advance.

9. Can we predict issue resolution time based on session conditions?
Build a model to estimate how long it will take to resolve an issue — helping dispatchers prioritize and plan better.

10. Can workers be segmented into performance groups to identify who needs support?
Use clustering to group workers by productivity, failure rate, and resolution efficiency — then recommend targeted interventions.
